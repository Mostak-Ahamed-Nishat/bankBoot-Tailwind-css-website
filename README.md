## Installation

```bash
git clone https://github.com/Mostak-Ahamed-Nishat/bankBoot-Tailwind-css-website.git
```
```bash
cd BankingBoot
```

```bash
npm i
```
```bash
npm run build

npm run watch
```
Run it on your browser
